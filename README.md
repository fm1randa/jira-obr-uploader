# Jira OBR Uploader

Uploads many OBRs at once to any Jira Server :fire:

## Run Locally

Clone the project

```bash
  git clone https://github.com/fm1randa/jira-obr-uploader
```

Go to the project directory

```bash
  cd jira-obr-uploader
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
